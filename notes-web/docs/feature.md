# 功能规划

## 当前状态

- **核心能力**: 只读浏览 — 7 种视图 (Table / Grid / List / Kanban / Timeline / Chart / Tree) + 全局搜索
- **架构**: Schema-driven + Pinia + 3 条泛型路由
- **打开方式**: 抽屉 / 弹窗 / 新页面，持久化到 localStorage
- **视图推断**: 根据 `FieldHints` 自动启用相关视图

---

阻碍日常使用的关键功能。

## P1 — 用户体验提升

体感差距较大的功能。

### 4. Markdown 实时编辑器

当前 `MdRenderer` 只负责渲染，无编辑能力。

- [ ] 分屏模式 (左编辑 + 右预览)
- [ ] 所见即所得模式
- [ ] 建议引入 CodeMirror 或 Monaco

### 5. 侧边栏折叠

`STORAGE_KEYS.SIDEBAR_COLLAPSED` 已定义未实现。

- [ ] 折叠按钮 + 动画过渡
- [ ] 折叠态 hover 展开

### 6. 全局错误/成功提示

当前 API 失败只有 `console.error`，用户无感知。

- [ ] 统一 `ElMessage` 错误提示
- [ ] 请求成功/失败/加载状态统一管理
- [ ] 网络断线检测

### 7. 搜索结果高亮

搜索结果列表没有匹配文本高亮。

- [ ] 标题/正文中匹配词高亮
- [ ] 返回匹配片段上下文

### 8. 字段筛选值动态获取

`GenericViewContainer` 的 `getFilters` API 已封装但从未调用。

- [ ] 筛选下拉框加载实际可选值
- [ ] 多字段联合筛选

### 9. 页面标题设为实际名称

路由守卫只设置 `document.title` 为路由名。

- [ ] 列表页标题: 显示类型中文名 (如"书籍")
- [ ] 详情页标题: 显示笔记标题

## P2 — 增强型特性

让工具更好用的进阶功能。

### 10. 多标签页管理

支持同时打开多条笔记详情，类似 IDE 标签页。

- [ ] 标签页栏
- [ ] 关闭/切换/右键菜单
- [ ] 持久化已打开标签

### 11. 标签云 / 全局标签筛选

FieldHints 有 `tags` 类型，但没有全局标签聚合。

- [ ] 标签云组件
- [ ] 跨类型标签聚合
- [ ] 点击标签过滤所有笔记

### 12. 详情页翻页导航

详情页只能关闭后回到列表选下一条，不支持快捷翻页。

- [ ] "上一篇 / 下一篇" 按钮
- [ ] 键盘方向键翻页

### 13. 笔记关联 / 反向链接

笔记之间通过 `link` 字段引用，但没有自动反向链接面板。

- [ ] 自动汇总引用了当前笔记的其他笔记
- [ ] "引用"面板显示在详情页
- [ ] 关联图谱可视化 (可选)

### 14. 全文搜索增强

目前只搜标题 + 正文。

- [ ] 扩展到字段值
- [ ] 搜索建议/自动补全
- [ ] 搜索历史记录

### 15. 浏览历史

没有浏览历史，无法回到刚才看过的笔记。

- [ ] 最近访问列表
- [ ] 时间倒序展示
- [ ] 一键清除历史

### 16. 全局快捷键

- [ ] `/` 聚焦搜索框
- [ ] `v` 切换视图模式
- [ ] `Esc` 关闭弹窗/抽屉
- [ ] `Ctrl+D` 切换深色模式
- [ ] `→/←` 详情页翻页

### 17. 笔记统计增强

- [ ] 字数统计
- [ ] 阅读时长预估
- [ ] 创建/修改时间线

### 18. 内容导出

- [ ] 单篇笔记导出为 Markdown 文件
- [ ] 批量导出为 ZIP
- [ ] 导出为 PDF (可选)

### 19. 对比模式

- [ ] 并排对比两条笔记
- [ ] 高亮差异字段

## 建议迭代路线

```
第 1 轮: 深色模式 + 侧边栏折叠 + 全局错误提示
  → 3 个低复杂度功能，立竿见影提升体感

第 2 轮: 笔记编辑 (CRUD) + Markdown 编辑器
  → 从"只读"升级为"可用"的笔记工具

第 3 轮: 看板持久化 + 筛选值动态获取 + 详情页翻页导航
  → 补齐交互断头路

第 4 轮: 标签聚合 + 历史记录 + 搜索结果高亮
  → 内容发现和信息检索

长期: 多标签页、笔记关联、测试、PWA、服务端分页
```
